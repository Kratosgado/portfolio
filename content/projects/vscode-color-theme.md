## Overview

VsCode Color Theme is a custom Visual Studio Code theme that merges the aesthetics of the GitHub Dark Default theme, the Andromeda theme, and VS Code's native Dark+ theme.

## Features

- **Unique Aesthetics**: A blend of popular dark themes.
- **Enhanced Contrast**: Improved visibility for coding.
- **Full Theme Coverage**: Applies to editor and UI elements.

## Technologies Used

- **JSON**: Configuration format for themes.
- **VS Code Theme**: Visual Studio Code's theming engine.
