## Overview

Flutter Demo Chat App is a messaging application built with Flutter for the frontend and powered by Firebase for backend services. It demonstrates real-time chat functionality and user authentication.

## Features

- **Real-Time Chat**: Send and receive messages instantly.
- **User Authentication**: Secure sign-up and login with Firebase Auth.
- **Backend Services**: Uses Firebase for database and authentication.

## Technologies Used

- **Flutter**: Google's UI toolkit for building beautiful, natively compiled applications.
- **Dart**: A client-optimized language for fast apps on any platform.
- **Firebase**: Google's mobile platform that helps you quickly develop high-quality apps.
