## Overview

Traffic Control App is an Android application that enables communication with an Arduino-based traffic light system via Bluetooth. It allows users to control the traffic lights remotely from their smartphones.

## Features

- **Bluetooth Connectivity**: Connects to the traffic light system.
- **Remote Control**: Change traffic light states from the app.
- **Status Monitoring**: View the current state of the traffic lights.

## Technologies Used

- **Android (Java)**: Native Android development.
- **Bluetooth**: Wireless communication technology.
