## Overview

AutoScroll Web Extension is a project that automatically scrolls the page for you.

## Features

- **Automated Scrolling**: Hand-free scrolling experience.
- **Configurable**: Settings to adjust scroll speed and behavior.

## Technologies Used

- **JavaScript**: Core logic implementation.
- **Web Extension API**: For browser integration.
