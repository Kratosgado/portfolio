## Overview

Instagram Clone is a mobile application built with React Native and TypeScript that mimics the core features and design of the popular social media platform, Instagram.

## Features

- **User Interface**: Replicates the Instagram UI for a familiar experience.
- **Photo Feed**: Scroll through a feed of posts.
- **User Interactions**: Like and comment on posts (simulated).

## Technologies Used

- **React Native**: A framework for building native apps using React.
- **TypeScript**: Typed JavaScript at Any Scale.
