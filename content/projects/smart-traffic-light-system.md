## Overview

Smart Traffic Light System is an IoT project that uses Arduino and an Android application to manage traffic lights. The system allows remote control and monitoring of traffic signals.

## Features

- **Traffic Light Control**: Manage red, yellow, and green lights.
- **Remote Operation**: Control the system using an Android app.
- **Hardware Integration**: Interfaces with Arduino for hardware control.

## Technologies Used

- **Arduino (C++)**: For microcontroller programming.
- **Android (Java)**: For the mobile control application.
